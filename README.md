# 🛡️ DFIR Quick Live Response Tool

**Windows Live Response batch script** used to collect important system information during a **Digital Forensic** or **Incident Response** investigation.

This tool gathers live system artifacts to assist investigators in performing quick triage analysis.

---

## ⚙️ Features

* ✅ System information collection
* 👤 Logged-in users enumeration
* 📊 Running processes listing
* 🌐 Network connections information
* 📅 Scheduled tasks enumeration
* 🔐 Persistence-related registry keys
* 🧩 Services information
* 🔥 Firewall status
* 💽 Disk and environment details

📄 All collected data is saved into a single output report.

---

## 🚀 Usage

### 1️⃣ Download the repository as a zip

### 2️⃣ Navigate to the downloaded zip file

### 3️⃣ Extract the content of the zip file

### 4️⃣ Run the script (.bat) **as Administrator** ⚠️

```
QUICK_LIVE_RESPONSE.bat
```

### 5️⃣ View Results 📁

The results will be generated automatically in the same directory:

```
Result.txt
```
